# Create books for the identity project
`type:task` · `area:identity` · `status:todo`
by [Doug](https://github.com/DNA-Platform/inexplicable-phenomena/blob/main/library/identity/doug/inexplicable-yet-self-evident/cover.md)

This repository is a **public library**, and **identity** is one of its subjects. A subject is known only through its **books** — and a book is where a subject's data is catalogued: what the subject holds, how it's organized, and how a reader moves through it. Today **identity** is a folder with no books, so there is nothing yet to read.

Write **identity**'s books — the ones that express how its data is catalogued, so the subject can be read the way any book in the library is.

Done when: **identity** has books that express how its data is catalogued.
