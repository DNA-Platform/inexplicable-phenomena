# Chemistry concepts

Pages explaining the deeper ideas the framework is built on — Self's prototype delegation, Scheme's environments, the `$` membrane, the dual constructor, prototypal shadowing, the chemistry metaphor.

Concept pages use the same template as features ([feature template]) with `kind: concept` in frontmatter. The form is the same — *what it is, how it works, related* — but a concept page describes a model rather than an API.

The existing entry-tier docs ([overview], [glossary]) cover most concepts at a high level today. Per-concept pages are written here as their depth justifies it.
