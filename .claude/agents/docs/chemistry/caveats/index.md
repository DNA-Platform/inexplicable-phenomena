# Chemistry caveats

One page per pitfall, gotcha, or hard-won lesson. Caveats are durable, linkable, named — they earn URLs because they get cited.

Use the [caveat template] when adding a new page.

When a caveat is resolved (the framework changes), set `status: resolved` or `status: historical` in the frontmatter. Don't delete it — the page is part of the project's institutional memory.
