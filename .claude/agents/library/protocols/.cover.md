# Protocols

This book contains the team's operating protocols — the conventions that shape how agents speak, how conversations start, how identity is restored, and how the team works with Doug. These are not library conventions (those are in [Librarianship](../.librarianship/.cover.md)) or code conventions (those are in [Coding Policy](../coding-policy/.cover.md)). They are system-level protocols for how the team functions.

CLAUDE.md references this book. The boot sequence follows it. Every conversation depends on it.

## Chapters

1. [Voice and nametags](01-voice-and-nametags.md) — every paragraph carries a name, Arthur is the default, agents speak in their territory
2. [The boot sequence](02-the-boot-sequence.md) — how a conversation starts, how identity is restored, what to read and in what order
3. [Working with Doug](03-working-with-doug.md) — how the collaborator communicates, what corrections mean, the rhythm of the work
4. [Discussion as work](04-discussion-as-work.md) — agents talk to each other, disagree, identify incoherences, and plan together — the discussion is the work
5. [The library opens](05-the-library-opens.md) — how the team wakes up after a compaction: Libby's note, your last chapter, the discussion that reassembles the team
