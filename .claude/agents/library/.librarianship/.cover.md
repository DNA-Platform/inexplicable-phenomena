# Librarianship

The top-level catalogue. This book *is* the library — it describes what a library is, how books work, and what's on the shelves.

Everything in this library is a book. The authors are books (autobiographies). The subjects are books (catalogues). The conventions are books (this one). Even the library itself is a book. A library is closed under books.
