:: The reference conundrum
:: The representation refers to the referent through its representative for it
:: We can infer that the representative is a representation of the referent
:: We can infer that the representation is representative of the referent
:: Let us suppose that the most specific way the reprsentative relates to the referent is through the representation
:: One perspective is that a reference refers to a referent because it is a type of representative in a representation
:: Another perspective is that the reference refers to the referent in the most fundamental way
:: It will be the subject of the reference frame that determines which of these is taken as the fundamental perspective 
representative -> represenntation -> reference
represenntation =representative> referent
representative =represenntation> referent
representative =represenntation> referent
(representative, referent) == represenntation
